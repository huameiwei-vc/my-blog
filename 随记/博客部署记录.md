## 部署流程 🚀

### 1. 初始化项目
```bash
# 在项目根目录初始化 git
git init

# 添加所有文件
git add .

# 提交更改
git commit -m "初始化项目"

# 创建并切换到 main 分支
git branch -M main

# 添加远程仓库
git remote add origin https://github.com/huameiwei-vc/my-blog.git

# 推送到 GitHub
git push -u origin main
```

### 2. Vercel 部署
1. 登录 Vercel 平台
2. 导入 GitHub 项目
3. 配置部署选项
4. 设置自定义域名

## 日常更新流程 🔄

### 方式一：命令行
```bash
cd my-blog
git add .
git commit -m "更新内容"
git push
```

### 方式二：一键脚本
> 使用自定义脚本实现一键更新（推荐）
```bash
~/Documents/cursor程序员/scripts/blog上传.sh
```
