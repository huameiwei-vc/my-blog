## 部署

```
# 在项目根目录初始化 git
git init

# 添加所有文件
git add .

# 提交更改
git commit -m "初始化项目"

git branch -M main

# 添加远程仓库（替换为您的 GitHub 仓库地址）
git remote add origin https://github.com/huameiwei-vc/my-blog.git

# 推送到 GitHub
git push -u origin main
```
